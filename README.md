# Match3

## Project
The program is an implementation of match 3 game using PyQt. User can choose board size and block types number.

## Requirements
Application uses Python 3.7

Libraries:
- PySide2
- random
- time
- copy
- sys

## Setup
```
git clone https://github.com/cytrynowyPiotras/match3PyQt.git
cd match3PyQt
python main.py
```
